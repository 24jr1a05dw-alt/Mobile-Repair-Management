# 📱 Dad's Mobile Repair Manager

A responsive Mobile Repair Shop Management System built using HTML, CSS, and JavaScript.

This project was created as a Father's Day gift to help manage repair jobs, customers, inventory, billing, and daily shop activities in a simple and professional way. :contentReference[oaicite:0]{index=0}

## 🌐 Live Demo

🔗 https://24jr1a05dw-alt.github.io/Mobile-Repair-Management/

## ✨ Features

- 🔧 Repair Job Management
- 🎫 Unique Repair Ticket Numbers
- 👥 Customer Search & History
- 📅 Due Today Deliveries
- 📦 Inventory Manager
- 🧾 PDF Bill Generator
- 📲 WhatsApp Customer Updates
- 🟡 Pending Status Tracking
- 🔵 Under Repair Status Tracking
- 🟢 Ready Status Tracking
- ✅ Delivered Status Tracking
- 💰 Earnings Dashboard
- 💾 Backup & Restore Data
- 🌙 Dark Mode
- 🌐 Telugu / English Support
- 📱 Progressive Web App (PWA)
- 📶 Offline Support
- ❤️ Family Corner

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage
- Service Workers
- PWA Manifest
- jsPDF

## 📱 Installation

1. Clone the repository
2. Open the project folder
3. Run with Live Server or deploy to GitHub Pages
